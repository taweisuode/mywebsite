参与ppf的开发工作¶

ppf目前是只有我来维护的项目，但它会接受大家贡献的文档和代码。 大家可以通过 github 上的 ppf 代码仓库 的 Issues 或者 Pull Requests来发起代码合并。

分支

ppf目前使用develop分支来进行开发的， 所以要求所有的 pull request 应该提交到 develop 分支,master分支上保持最正确的代码
