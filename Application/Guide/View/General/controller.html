控制器¶

什么是控制器？

简而言之，一个控制器就是一个类文件，是以一种能够和 URI 关联在一起的方式来命名的。
查看下面的URI

                            
    http://jingpeng.blog.com/Guide/General/controller
  • Guide为模块名
  • General为控制器名
  • controller为方法名

如何创建控制器？

                            
<?php

/**
 * @desc ppf框架用户手册
 */
    class GeneralController extends Controller {
        public function controllerAction() {
            $this->view->show();
        }
    }
?>

根据以上实例，我们能够很方便的创建controller类

$this->view->show();

是默认指向 View/控制器名下面的视图文件

如何在控制器类中使用构造函数？

可以查看该用户手册的开源项目地址并下载查看

例如本控制器中的构造函数有如下的写法


 <?php

/**
 * @desc ppf框架用户手册
 */
    class GeneralController extends Controller {
        public function __CONSTRUCT() {
            parent::__CONSTRUCT();
            $directoryModel = new DirecotryManageModel();
            $main_node_arr = $directoryModel->directoryData();
            $this->view->assign('main_node_arr',$main_node_arr);
        }

    }
?>

提示

可以看出如果需要调用子类的构造方法，必须先调用父类的构造函数
