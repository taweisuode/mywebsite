视图¶

什么是视图？

简单来说，一个视图其实就是一个 Web 页面，或者页面的一部分，像页头、页脚、侧边栏等。
我们可以通过include的方式包括进来，这样就减少了代码量，并且能够更加方便的处理这些公共页面

使用在 控制器 页面中创建的控制器例子，让我们再添加一个视图。 

                            
    <html>
    <head>
    <title>ppf 用户手册</title>
    </head>
    <body>
    <h1>hello ppf!</h1>
    </body>
    </html>

将这个文件保持到View目录下对应的Controller名称的目录下

在控制器中指定对应的视图文件

$this->view->show($file_name);

如何在视图中使用变量字符串，数组等或者使用一些逻辑判断？

可以参考smarty的视图表示变量的表达式，ppf也是使用类似的模版引擎来做视图的变量解析，如下所示

                            
<html>
<head>
<title>{\$result}</title>
</head>
<body>
{\if $max == 4}
aaa
{\elseif $max == 6}
asaas
{\else}
ccc
{\/if}
<table>
{\foreach name=$fruit item=key value=val}
<tr>
<td>{\$key}</td><d>{\$val}</td>
</tr>
{\/foreach}
<table>
</body>
</html>

提示

切记！！，上面的demo需要将所有的 \符号去掉（由于模版引擎的原因，他会被解析）

可以根据视图文件，我们需要在控制器类中传递参数给视图文件


    <?php
    public function controllerAction() {
        $result = "hello ppf!";
        $arr = array("test"=>"aaa","hello"=>"bbb");
        $this->view->assign('result',$result);
        $this->view->assign('arr',$arr);
        $this->view->show();
    }
    ?>
